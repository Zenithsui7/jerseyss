## Assets

All CSS, JS, and image assets are in the `assets/` folder, organised by page:

- `assets/home/` — Homepage assets
- `assets/half-sleeve/` — Half Sleeve page assets
- `assets/full-sleeve/` — Full Sleeve page assets
- `assets/contact/` — Contact page assets
- `assets/shipping-policy/` — Shipping Policy page assets

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Framework Preset: **Other** (Static)
5. Root Directory: `.` (leave as default)
6. Click **Deploy**

No build command needed — this is a plain static site.
